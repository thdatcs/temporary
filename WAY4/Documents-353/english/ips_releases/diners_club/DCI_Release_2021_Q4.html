DCI Requirements Report for 2021.Q4 release

This document contains requirements for DCI release 2021.Q4. For each requirement, its implementation status and the versions of WAY components implementing the requirement are shown (to comply with DCI requirements, the specified version of the WAY4 component or a newer version must be installed at the bank (processing centre)).

The DCI requirement statuses shown in the report indicate the following:

  • [IN ANALYSIS] - At the time of the report's generation, analysis of the DCI requirement's impact on WAY4 (software code) is under way.
  • [WAIT] - The next version of release documents is expected.
  • [IN PROGRESS] - Development is under way at the time of the report's generation. Development has not been completed. The list of builds for this requirement is incomplete and will possibly be updated.
  • [SUPPORTED] - WAY4 (software code) has been changed to comply with the DCI requirement. The list of builds for this requirement is complete. The list of builds may be updated if an updated requirement is received from the DCI , or if implementation errors are found (the status of the requirement itself does not change).
  • [NOTHING TO CHANGE] - At the time of the report's generation, the impact of the DCI requirement on WAY4 has been analysed. A decision not to change WAY4 (software code) was made. Requirement already up-to-date.
  • [NOT SUPPORTED] - At the time of the report's generation, the impact of the DCI requirement on WAY4 has been analysed. A decision not to change WAY4 (software code) was made.
  • [SUPPORTED TECHNICALLY] - Task status indicating that the release requirement is only supported technically.
